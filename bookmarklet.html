Step 2 — Install on your device

Installation instructions

Safari on iPhone does not allow dragging bookmarks. You must create a bookmark first, then edit its URL.
  1. Copy the bookmarklet code — tap the Copy button above.
  2. Open Safari and go to any page — e.g. wayfarer.nianticlabs.com.
  3. Bookmark the current page — tap the Share button ⎙ then Add Bookmark. Name it S2 Wayfarer and tap Save. Share → Add Bookmark → Save
  4. Edit the bookmark URL:
    1. Tap the book icon (Bookmarks) at the bottom of Safari.
    2. Find S2 Wayfarer, then long-press → Edit.
    3. Tap the Address field, select all (triple-tap), and paste the copied code.
    4. Tap Done.
  5. Use it on Wayfarer:
    1. Navigate to wayfarer.nianticlabs.com and open the map view.
    2. Tap the address bar and type S2 Wayfarer — your bookmark will appear.
    3. Tap it. The S2 Cells panel will appear in the top-right corner.
    Address bar → type "S2" → tap bookmark suggestion
Chrome on Android does not allow dragging bookmarks. Create a bookmark first, then edit it.
  1. Copy the bookmarklet code — tap Copy above.
  2. Bookmark any page — open Chrome, tap the ⋮ menu → Star / Bookmark this page. Name it S2 Wayfarer and save. ⋮ → Bookmark this page → Edit name → Save
  3. Edit the bookmark URL:
    1. Tap the address bar, type S2 Wayfarer, then long-press the bookmark → Edit.
    2. Clear the URL field and paste the copied javascript:… code.
    3. Tap Save.
  4. Use it on Wayfarer:
    1. Go to wayfarer.nianticlabs.com and open the map.
    2. Tap the address bar, type S2, and tap the S2 Wayfarer bookmark.
    3. The floating S2 panel will appear.

Method A — Drag (easiest)

  1. Make sure your bookmarks bar is visible:
    • Chrome / Edge: Ctrl+Shift+B (Windows) or ⌘+Shift+B (Mac)
    • Firefox: Ctrl+Shift+B → View → Toolbars → Bookmarks Toolbar
  2. Drag the ⬡ S2 Wayfarer button at the top of this page to your bookmarks bar.
  3. Navigate to wayfarer.nianticlabs.com, open the map view, and click the bookmark.

Method B — Manual (if drag doesn't work)

  1. Copy the javascript:… code from the box above.
  2. Right-click your bookmarks bar → Add page… (Chrome) or New Bookmark (Firefox).
  3. Set Name to S2 Wayfarer and paste the code into the URL field.
  4. Save and click the bookmark while on Wayfarer.
What the bookmarklet does

Overlay behaviour

  • Detects the Google Maps instance already running on Wayfarer — no external requests.
  • Draws L14 cells as thick blue polygons (gym logic cells).
  • Draws L17 cells as thin yellow polygons (individual stop spacing).
  • Updates automatically on every pan and zoom via the Google Maps idle event.
  • A small floating panel (top-right corner) lets you toggle each level on/off or dismiss it.
  • Clicking the bookmarklet a second time re-shows the panel if you dismissed it.
  • Cells only render when the zoom is useful — L14 at zoom ≥ 11, L17 at zoom ≥ 14.
  • All S2 geometry is computed locally inside the bookmarklet — no network calls.

Step 3 — Verify the overlay is correct

How to verify cell accuracy

Compare your overlay against vesoyu.github.io/s2cell:

  1. Open vesoyu.github.io/s2cell in a separate tab and navigate to the same location.
  2. Enable L14 on vesoyu's map. The thick blue grid should match exactly.
  3. Enable L17. The thin yellow lines should match perfectly.
  4. Pan and zoom — the grid updates within 1–2 seconds after you stop moving.

Common problems & fixes

  • Panel says "Map not found" — the bookmarklet couldn't detect the Google Maps instance. Wait for the map to fully load, then click the bookmark again.
  • No cells visible — check zoom level. L14 appears at zoom 11+, L17 at zoom 14+.
  • Panel is hidden — click the bookmarklet again to toggle the panel back on.
  • Cells drift after panning — normal; the grid redraws on each idle event (~0.5s after stopping).

Bookmarklet is 100% client-side. No data is sent anywhere. S2 geometry computed locally.
