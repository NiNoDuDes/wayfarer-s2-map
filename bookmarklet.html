Step 2 — Install on your device

Installation instructions

Safari on iPhone does not allow dragging bookmarks. You must create a bookmark first, then edit its URL.
  1. Copy the bookmarklet code — tap the Copy button above to copy the full javascript:… text to your clipboard.
  2. Open Safari and go to any page — for example, open wayfarer.nianticlabs.com.
  3. Bookmark the current page — tap the Share button ⎙ at the bottom of Safari, then tap Add Bookmark. Name it S2 Wayfarer and tap Save. Share → Add Bookmark → Save
  4. Edit the bookmark URL:
    1. Tap the book icon (Bookmarks) at the bottom of Safari.
    2. Find S2 Wayfarer, then long-press → Edit.
    3. Tap the Address field, select all (triple-tap), and paste the copied code.
    4. Tap Done.
  5. Use it on Wayfarer:
    1. Navigate to wayfarer.nianticlabs.com and open a nomination so the map is visible.
    2. Tap the address bar and type S2 Wayfarer — your bookmark will appear in suggestions.
    3. Tap it. The S2 Cells panel will appear in the top-right corner of the map.
    Address bar → type "S2" → tap bookmark suggestion
Chrome on Android does not allow dragging bookmarks. Create a bookmark first, then edit it.
  1. Copy the bookmarklet code — tap Copy above.
  2. Bookmark any page — open Chrome, tap the ⋮ menu → Star / Bookmark this page. Name it S2 Wayfarer and save. ⋮ → Bookmark this page → Edit name → Save
  3. Edit the bookmark URL:
    1. Tap the address bar, type S2 Wayfarer to surface the bookmark, then long-press the bookmark → Edit.
    2. Clear the URL field and paste the copied javascript:… code.
    3. Tap Save.
  4. Use it on Wayfarer:
    1. Go to wayfarer.nianticlabs.com and open a nomination map.
    2. Tap the address bar, type S2, and tap the S2 Wayfarer bookmark.
    3. The floating S2 panel will appear.

Method A — Drag (easiest)

  1. Make sure your bookmarks bar is visible:
    • Chrome / Edge: Ctrl+Shift+B (Windows) or ⌘+Shift+B (Mac)
    • Firefox: Ctrl+Shift+B → View → Toolbars → Bookmarks Toolbar
  2. Drag the ⬡ S2 Wayfarer button at the top of this page to your bookmarks bar.
  3. Navigate to wayfarer.nianticlabs.com, open a nomination, and click the bookmark.

Method B — Manual (if drag doesn't work)

  1. Copy the javascript:… code from the box above.
  2. Right-click your bookmarks bar → Add page… (Chrome) or New Bookmark (Firefox).
  3. Set Name to S2 Wayfarer and paste the code into the URL field.
  4. Save and click the bookmark while on Wayfarer.
What the bookmarklet does

Overlay behaviour

  • Detects the Google Maps instance already running on Wayfarer — no external requests.
  • Draws L14 cells as thick blue polygons (gym logic cells).
  • Draws L17 cells as thin yellow polygons (individual stop spacing).
  • Updates automatically on every pan and zoom via the Google Maps idle event.
  • A small floating panel (top-right corner) lets you toggle each level on/off or dismiss it.
  • Clicking the bookmarklet a second time re-shows the panel if you dismissed it.
  • Cells only render when the zoom is useful — L14 at zoom ≥ 11, L17 at zoom ≥ 14.
  • All S2 geometry is computed locally inside the bookmarklet — no network calls, works offline.

Gym count rules (per L14 cell)

Stops in L14 cellGymsNext gym trigger
0 – 10Add 2 stops total
2 – 51Add 6 stops total
6 – 192Add 20 stops total
20+3—

Gym eligibility is determined by the number of approved Wayspots (stops + gyms) inside the same L14 S2 cell. When a new Wayspot is approved in a cell that crosses a threshold, the oldest or most-interacted-with stop becomes a gym.

Step 3 — Verify the overlay is correct

How to verify cell accuracy

Compare your overlay against the reference tool vesoyu.github.io/s2cell:

  1. Open vesoyu.github.io/s2cell in a separate tab. Navigate to the same location as your Wayfarer map.
  2. Enable L14 on vesoyu's map. The thick blue grid lines on Wayfarer should line up exactly with vesoyu's L14 boundaries.
  3. Enable L17 on vesoyu's map. The thin yellow lines on Wayfarer should match perfectly.
  4. Pan and zoom on Wayfarer — the grid should update within 1–2 seconds after you stop moving (idle event).

Common problems & fixes

  • Panel says "Map not found" — the bookmarklet was run before the Wayfarer map fully loaded. Wait for the map to appear, then tap the bookmark again.
  • No cells visible at all — check you are zoomed in enough. L14 cells appear at zoom 11+, L17 at zoom 14+. On Wayfarer the nomination map is typically at zoom 15–17.
  • Panel is hidden — click / tap the bookmarklet again to toggle the panel back on.
  • Cells drift slightly after panning — this is normal; the grid redraws on each idle event (≈0.5 s after you stop). There is no visual lag mid-pan by design, keeping the map usable.
  • Wayfarer updated and the map broke — the bookmarklet finds the Google Maps instance by scanning DOM element properties. If Niantic changes the page structure significantly this can stop working. Re-run from this page — an updated version will be here.

Bookmarklet is 100% client-side. No data is sent anywhere. S2 geometry computed locally.
